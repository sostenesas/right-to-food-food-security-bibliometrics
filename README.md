# Right to Food & Food Security — Bibliometric Project

This repository contains code and documentation for a bibliometric study on:
- the Human Right to Adequate Food
- Food Security / Food Insecurity

Primary data source: OpenAlex.

## Project status
This repository is under active development.

## Structure (initial)
- protocol/  – search strategy and inclusion criteria
- data/      – raw and processed data (not tracked by git)
- R/         – R scripts
- outputs/   – figures and tables
